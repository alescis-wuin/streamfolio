Les captures complémentaires sont placées dans un bloc repliable pour ne pas alourdir la lecture du README.

## GIF court optionnel

Le GIF long actuel est stocké dans :

```text
docs/record/demo.gif
```

Pour un README public, ajouter plus tard un GIF court dans :

```text
docs/screenshots/demo.gif
```

Cible recommandée : 15 à 25 secondes, 10 à 12 FPS, moins de 10 Mo si possible.
