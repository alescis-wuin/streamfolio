# Captures d'écran

Captures recommandées pour le README et le portfolio.

## Où les prendre

1. Lancer l'application :

```bash
cd backend
mvn spring-boot:run
```

2. Ouvrir `http://localhost:8080`.
3. Se connecter avec le compte de démonstration : `alexis@example.dev / demo1234`.

## Fichiers à placer ici

| Fichier | Écran à capturer | Format conseillé |
|---|---|---|
| `home.png` | Accueil après connexion, avec hero et rails visibles | 1440 × 900 |
| `detail.png` | Fiche détail d'un film ou d'une série | 1440 × 900 |
| `player.png` | Lecteur vidéo en cours de lecture, sous-titres visibles si possible | 1440 × 900 |
| `mobile.png` | Accueil ou fiche détail en responsive mobile | 390 × 844 |
| `demo.gif` | Parcours court : login → recherche → détail → lecture → watchlist | 15 à 30 s, idéalement < 10 Mo |

## Insertion dans le README

Une fois les fichiers ajoutés, tu peux remplacer la section `Captures d'écran` du README par exemple par :

```md
![Accueil Streamfolio](docs/screenshots/home.png)

![Lecteur vidéo](docs/screenshots/player.png)
```

Ou, pour un GIF :

```md
![Démo Streamfolio](docs/screenshots/demo.gif)
```
